How to Redeem a Promo Code

  1. Open the App Store

  2. Tap on your profile icon located at the top right corner

  3. Tap on "Redeem Gift Card or Code"

  4. Tap "Enter Code Manually"

  5. Enter the provided code and tap the "Redeem" button

  6. Enter your Apple password if requested and then the app will start downloading

  7. You will see a message that the code was redeemed successfully
